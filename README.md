# ☁ Cloudview

Cloudview is a modern and responsive weather web application built using HTML, CSS, and JavaScript. It allows users to search for any city and get real-time weather updates using a live weather API.

## 🌍 Features

- 🔎 Search weather by city name
- 🌡 Real-time temperature display
- 💧 Humidity information
- 🌬 Wind speed details
- ☁ Weather condition status
- 📱 Responsive design for mobile and desktop
- ⚡ Fast and simple user interface

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Weather API

## 🚀 How to Use

1. Enter a city name
2. Click the Search button
3. View live weather details instantly

## 📂 Project Structure

Cloudview/
│── index.html
│── style.css
│── script.js

## 🎯 Purpose

This project was created to practice frontend web development skills, API integration, and responsive UI design.
